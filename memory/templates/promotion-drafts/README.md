# Promotion Drafts

This folder stores generated draft artifacts promoted from `LEARNING_BACKLOG.md`.

It is meant for review, not automatic activation.

Recommended usage:

1. Generate drafts with `scripts/generate-superpowers-promotion-drafts.ps1`
2. Review the generated checklist, rule, or skill draft
3. Edit the draft as needed
4. Only then promote it into a real workflow, project rule, checklist, or skill

Suggested subfolders:

- `checklists/`
- `rules/`
- `skills/`

Keep generated drafts small, reviewable, and tied back to their original candidate IDs.

### P3: 智能推荐实现

#### 3.1 GO.md集成

**当前状态**: 算法已实现（mode-recommender.py），但未集成到GO.md

**待完成**:
- [ ] 在GO.md第五步调用Python脚本
- [ ] 或改用Markdown实现的简化版
- [ ] 创建MODE_HISTORY.md模板
- [ ] 实现自动记录机制

**方案A: Python集成**（推荐）
```markdown
## 第五步 · 模式判定

### 5.1 智能推荐

如果 `.specs/MODE_HISTORY.md` 存在且有≥10条记录：

```bash
python scripts/mode-recommender.py --requirement "<用户需求>"
```

输出推荐结果，等用户确认。

### 5.2 规则判定（fallback）

如果无历史数据，使用原有规则。
```

**方案B: Markdown实现**（简单）
```markdown
## 第五步 · 模式判定

### 快速评估

**需求长度**:
- <20字 → 倾向Fast
- 20-100字 → 倾向Standard
- >100字 → 倾向Strict

**关键词**:
- "修复/typo" → Fast
- "功能/页面" → Standard
- "支付/鉴权" → Strict

基于以上，推荐：**Standard**

请确认或选择其他模式...
```

**预计工作量**: 1周  
**优先级**: P2

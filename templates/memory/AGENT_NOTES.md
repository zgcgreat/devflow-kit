本文件用于记录AI助手端关于执行质量、仓库特定处理或重复操作陷阱的持久提醒。

## 条目模板

```md
### AI助手笔记：<简短标题>

- id: agent-YYYY-MM-DD-<标识>
- type: agent_note
- status: active
- confidence: verified
- last_updated: YYYY-MM-DD
- source:
- owner:
- review_after:

提醒内容：


重要性：


如何应用：

```

## 说明

- 与项目事实和用户偏好分开保存。
- 本文件用于可重复的执行提醒，而非临时草稿笔记。
- 如果笔记不再相关，标记为 `superseded` 而不是静默删除历史。
